* Declarative Agent Skills Module

This Home Manager module provides declarative management of rules and instructions ("skills") for AI coding agents across multiple directories. It supports fetching skills from GitHub repositories, indexing them in a local catalog, resolving name conflicts, and symlinking them to active agent environments.

** Target Agent Directories
- *Antigravity CLI*: =~/.gemini/antigravity-cli/skills=
- *Gemini CLI*: =~/.gemini/skills=
- *Claude Code*: =~/.claude/skills=
- *Cursor*: =~/.cursor/skills=
- *Codex*: =~/.agents/skills=

** Key Design Decisions:
1. *Deterministic & Offline-Safe Builds*: Pinned hashes and commit SHAs are locked in =skills-catalog.json=. This guarantees builds are 100% reproducible, offline-safe, and decoupled from network availability.
2. *Efficient Local Prefetching*: To avoid GitHub API rate limits, the update script fetches each repository archive once via =nix-prefetch-url= and parses the files locally inside the Nix store path.
3. *Cross-Repository Collision Resolution*: Selected skills are referenced as flat attributes. If a skill name collision occurs across different repositories (e.g. =tdd= in both Cursor and Matt Pocock's skills), the update script automatically prefixes them with the repository key (e.g., =cursor-tdd= and =pocock-tdd=) to prevent conflicts.

* 2. Configuration

** Import the Module
Add the directory path to your imports block inside [[file:///home/justin/dotfiles/home-manager/home.nix][home.nix]]:
#+begin_src nix
imports = [
  ./modules/agentic-skills
];
#+end_src

** Configure Settings
Enable the module and specify which skills should be enabled:
#+begin_src nix
modules.agentic-skills = {
  enable = true;
  skills = {
    frontend-design.enable = true;
    cursor-tdd.enable = true;
    pocock-tdd.enable = true;
    # Optional overrides:
    # other-skill = {
    #   enable = true;
    #   rename = "custom-name";
    #   agents = [ "cursor" ];
    # };
  };
};
#+end_src

* 3. Options Reference

| Option                                             | Type     | Default   | Description                                                        |
|----------------------------------------------------+----------+-----------+--------------------------------------------------------------------|
| =modules.agentic-skills.enable=                    | =bool=   | =false=   | Master toggle to enable skills symlinking.                         |
| =modules.agentic-skills.skills.<skillName>.enable= | =bool=   | =false=   | Set true to enable a specific skill. Autocompletes via LSP.        |
| =modules.agentic-skills.skills.<skillName>.rename= | =string= | =null=    | Custom name for the skill folder inside agent directories.         |
| =modules.agentic-skills.skills.<skillName>.agents= | =list=   | =[]=      | Agents to link this skill to. If empty ([]), links to all active.  |
| =modules.agentic-skills.agents.<name>.enable=      | =bool=   | =true=    | Installs symlinks to target agent =<name>=.                        |
| =modules.agentic-skills.agents.<name>.path=        | =string= | *(varies)*| Target relative directory path for the agent.                      |
